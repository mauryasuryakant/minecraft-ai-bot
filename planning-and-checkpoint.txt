PROJECT NAME
Minecraft AI Companion Bot

PROJECT PURPOSE
Build a lightweight AI-driven Minecraft bot that can understand
chat commands and perform automated tasks like mining, moving,
collecting resources, and following the player.

This bot is intended to become an intelligent in-game assistant.

--------------------------------------------------
AVAILABLE BOT COMMANDS
--------------------------------------------------

Prefix: !

1. Movement

!goto X Y Z -> Moves to coordinates.
!follow -> Follows the player.
!come -> Walks to the player's current position.
!guide [TARGET] -> Leads you to a target (village/block).

2. Mining & Collection

!mine BLOCK_NAME -> Mines the nearest block.
!collect BLOCK_NAME -> Collects nearby blocks (up to 5).
!stripmine DIRECTION [DISTANCE] -> Starts a 1x2 tunnel.
!scan BLOCK_NAME -> Finds the location of nearby blocks.
!findvillage -> Finds nearby village coordinates.

3. Survival & Utility

!status -> Shows health, hunger, and position.
!eat -> Manually triggers eating from inventory.
!sleep -> Finds and uses a nearby bed.
!drop -> Drops all items in inventory.
!stop -> Stops current actions.
